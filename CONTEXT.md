# ClawBridge Admin

ClawBridge Admin 是面向平台管理员的管理界面。本词汇表定义普通管理页面的产品形态，避免同类信息采用互不一致的布局语言。

## 页面形态

**普通管理页面**：
遵循统一管理布局语言的页面，包括卡片集合页、数据记录页和配置详情页。业务字段与流程不因布局统一而改变。
_Avoid_: 常规页面、标准页面

**卡片集合页**：
以独立实体为浏览单位的集合页面，强调每个实体的身份、摘要、状态和常用操作。Agent、Skill、知识库和订阅套餐属于此类。
_Avoid_: 卡片列表页、宫格页

**数据记录页**：
以跨行比较、排序和高密度扫描为主要目的的集合页面。用户、用量、会话、插件、审计日志和工具权限属于此类。
_Avoid_: 表格页、数据列表页

**配置详情页**：
围绕单个对象或系统领域组织信息、表单和分区操作的页面。Agent 详情、OpenClaw MCP、系统设置和安全策略属于此类。
_Avoid_: 设置页、详情表单页

**配置分区**：
配置详情页中具有独立语义和局部操作边界的内容区域。全页操作属于页面操作区，单个分区的操作属于对应配置分区。
_Avoid_: 设置卡片、表单卡片

**特殊页面**：
因信息架构或核心交互明显不同，不套用普通管理页面形态的页面。概览和模型属于此类。
_Avoid_: 例外页面

## 页面操作

**页面操作区**：
作用于整个当前页面的操作区域，包含刷新、导入、上传和新建。次级操作排列在左，主操作排列在最右；窄屏下可以图标化，过多的次要操作收纳到更多菜单，但区域保持单行。
_Avoid_: 标题栏按钮区、顶部按钮插槽

**内容工具栏**：
用于改变当前集合呈现结果的控制区域，包含已有的搜索、筛选、排序和视图切换能力。没有对应能力时不显示空工具栏；卡片集合页在桌面端左侧放筛选、右侧放搜索，移动端依次纵向排列。
_Avoid_: 搜索栏、筛选栏

**集合搜索**：
内容工具栏中用于缩小集合结果范围的单一搜索入口，具有一致的搜索提示和清除能力。数据来源可以决定即时过滤或提交查询，但不改变其页面形态。
_Avoid_: 搜索表单、关键词字段

**集合排序**：
内容工具栏中用于改变卡片集合顺序的可见控制。表格转换为卡片时保留原有排序能力，并将排序字段与方向表达为明确选项。
_Avoid_: 隐式排序、表头排序

## 集合卡片

**集合卡片**：
卡片集合页中代表单个实体的标准浏览单元。顶部呈现名称与标识，右上角呈现条目操作，中部呈现说明或标签，底部呈现次级元数据。
_Avoid_: 条目卡片、信息卡片

**卡片操作**：
位于集合卡片右上角、作用于当前实体的操作。常用操作可以直接显示，危险操作收纳在纯图标更多菜单中。
_Avoid_: 卡片底部操作、行操作

**主要状态**：
与实体名称共同构成身份识别的当前状态，位于卡片名称旁并保持完整显示。卡片右上角只用于卡片操作。
_Avoid_: 右上角状态、底部状态

**卡片元数据**：
位于集合卡片下方的版本、大小、来源、更新时间、额度等次级信息，不与实体名称和主要状态争夺视觉层级。
_Avoid_: 卡片详情、辅助字段

**卡片摘要**：
集合卡片中对长文本、重复配置或多项明细的紧凑表达。摘要保留管理判断所需的信息，完整内容仍由现有编辑流程或辅助提示承载。
_Avoid_: 精简字段、隐藏信息

**整卡入口**：
当实体存在独立详情页时，集合卡片整体作为进入详情的主要入口。卡片操作保持独立，触发操作时不进入详情页。
_Avoid_: 查看详情按钮、仅标题链接

**卡片集合**：
集合卡片组成的响应式内容区域，在窄屏、中等宽度和宽屏下分别使用一列、两列和三列。卡片数量不足时不拉伸填满整行。
_Avoid_: 卡片网格、宫格

## 集合状态

**集合空状态**：
内容工具栏之后、占满集合宽度的无结果说明。数据源为空与筛选无结果是不同状态，使用不同文案。
_Avoid_: 空卡片、空表格

**集合错误**：
集合加载或刷新失败时在集合顶部呈现的持续错误信息。刷新失败保留已有结果和用户输入。
_Avoid_: 错误空状态、仅 Toast 错误

**集合骨架**：
卡片集合首次加载时模拟最终列数和卡片高度的占位内容。
_Avoid_: 全页加载、通用旋转加载

**局部操作**：
只作用于单个条目或独立子区域的操作。条目操作属于对应条目，子区域刷新属于对应区域。
_Avoid_: 页面操作

**统计摘要**：
在集合内容之前呈现、用于辅助管理判断的关键指标区域。只有存在有决策价值的指标时才展示，不作为页面一致性的强制占位。
_Avoid_: 数据概览、统计卡片区

**页面标题**：
标识当前普通管理页面的唯一一级标题，位于顶部导航区域。内容区域不重复页面标题。
_Avoid_: 内容标题、页面主标题

**子视图标题**：
标识同一页面内部独立视图的标题，仅在页面包含多个可切换视图时出现，并可附带对当前视图有帮助的说明。
_Avoid_: 页面标题

**布局节奏**：
普通管理页面的固定间距层级。页面主要区域使用宽间距，工具栏、卡片集合和卡片内部使用中间距，表格、错误和分页等紧密相关内容使用紧间距。
_Avoid_: 页面密度、组件间距

**重复标题**：
在页面标题、视图切换或内容工具栏已经清楚表达当前内容后再次出现的同义标题。普通集合与表格不展示重复标题，例如用量视图下方不再重复“按日用量”或“按用户用量”。
_Avoid_: 区块标题

## 数据记录

**记录集合**：
数据记录页中的主要内容区域，由可选统计摘要、内容工具栏、记录表格和分页信息依次组成。表格不使用装饰性卡片包裹。
_Avoid_: 表格区、数据卡片

**记录操作**：
作用于单条记录的操作。常用操作可以直接显示，危险操作收纳在纯图标更多菜单中。
_Avoid_: 行按钮、表格操作

## OpenClaw

**OpenClaw 实例**：
一个通过 `openclaw-plugin-rpc` 连接到 ClawCore Bridge 的 OpenClaw 运行环境。
_Avoid_: RPC 插件、实例插件

**实例连接标识**：
由 OpenClaw 实例配置的稳定 `pluginId`，用于在 Bridge 中唯一选择当前 RPC 连接；它不是插件库中的插件标识。
_Avoid_: 插件 ID、实例 ID

**实例连接清单**：
ClawCore Bridge 当前可见的 OpenClaw 实例连接快照，不保留已经从连接注册表消失的历史实例。
_Avoid_: 实例资产库、离线实例历史

**控制动作已受理**：
目标 OpenClaw 实例已接受重启或重载请求，但尚未证明动作完成或实例恢复可用。
_Avoid_: 重启成功、操作完成

**重启恢复**：
重启请求受理后，实例以新的连接或运行时启动状态恢复在线。
_Avoid_: 在线、重启已受理

**插件**：
由 `pluginId` 标识的插件产品，包含一个或多个插件版本。
_Avoid_: 插件分组、插件包

**插件版本**：
插件在插件库中的一个版本记录，可被安装到 OpenClaw 实例。
_Avoid_: 插件、安装版本

**已删除插件版本**：
插件库中被软删除、仅供详情审计的历史版本记录，不再允许安装或编辑。
_Avoid_: 已下架插件、已删除插件

**已下架插件**：
没有可用插件版本但仍存在实例安装的插件。
_Avoid_: 已删除插件、不存在的插件

**实例安装**：
某个插件版本与一个 OpenClaw 实例之间的持久关系，包含安装状态、启用状态、生效范围和配置；卸载后关系仍然保留。
_Avoid_: 实例插件、插件实例

**活动实例安装**：
状态为安装中或已安装的实例安装，会阻止其插件版本被删除。
_Avoid_: 有效安装、正在使用的插件

**实例安装清单**：
跨插件、跨 OpenClaw 实例展示实例安装状态和错误的只读清单。
_Avoid_: 实例插件表格、实例管理页
